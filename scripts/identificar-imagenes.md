# Guía para Identificar y Renombrar Imágenes

Basándome en las descripciones de las imágenes que compartiste, aquí está el mapeo de platos:

## Proceso Manual Recomendado:

1. **Abre la carpeta `public/media/` en tu computadora**
2. **Revisa cada imagen visualmente**
3. **Renombra cada imagen según el plato que muestra**
